# Anulen Agency Website

A simple base website for Anulen, built with Next.js, TypeScript, and Tailwind CSS.

## Sections

- Navbar
- Hero
- Services preview
- Process
- Projects preview
- About
- Contact CTA
- Footer

## Setup

```bash
npm install
npm run dev
```

Then open:

```bash
http://localhost:3000
```

## Suggested Git workflow

```bash
git checkout -b stage-1-base-website
git add .
git commit -m "Build stage 1 base agency website"
git push -u origin stage-1-base-website
```
